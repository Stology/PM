```

│                             │                               │
│                             │   [우측 사이드 패널]            │
```

### 영역 비율
- 그래프 캔버스: 메인 영역
- 사이드 패널: 노드 클릭 시 우측에서 슬라이드 인 (약 30~35% 너비)
- 사이드 패널이 닫혔을 때는 그래프 전체 영역 사용

## 화면 구성 요소

### 1. 상단 필터 영역

#### 뷰 필터
- **A. 전체 활성 노드 보기** (기본)
- **B. 특정 주차 보기** (드롭다운에서 주차 선택)
  - 선택 주차에 신규 활성화된 노드와 보강된 노드를 모두 강조
  - 신규 활성 노드와 보강 노드는 색상, 배지, 선 스타일 등으로 구분
  - 나머지는 회색 반투명

#### 노드명 검색
- 검색 대상은 주차 필터와 무관한 온톨로지 전체 노드다.
- 사용자가 노드명을 입력하면 검색 후보 목록을 표시한다.
- 검색 결과 노드를 선택하면 해당 노드로 그래프를 이동/강조하고 우측 사이드 패널을 해당 노드 상세로 전환한다.
- 검색어가 없거나 검색 결과를 닫으면 기존 그래프/필터 상태를 유지한다.
- 검색 결과가 없으면 "일치하는 노드가 없습니다" 빈 상태를 표시한다.

### 2. 3D 그래프 캔버스 (메인 영역)

#### 노드 시각 표현

| 상태 | 시각 표현 |
|---|---|
| 비활성 | 회색 반투명 |
| 활성 | 단일 색상 (전체 노드 동일색) |
| 활성 강도 | 자료 수에 비례한 색상 진하기 |
| 강조 | 선택 노드와 연결된 노드들 (테두리 강조 등) |

> 활성화 강도는 단순 on/off가 아닌 단계적 표현. 자료가 많이 연결된 노드일수록 색상이 진해짐.

#### 엣지 (관계선)
- 단순 선 (방향성 없음)
- 별도 클릭 액션 없음

#### 그래프 조작
| 조작 | 동작 |
|---|---|
| 드래그 (빈 공간) | 그래프 회전 |
| 스크롤 | 줌 인/아웃 |
| 노드 드래그 | 일시 이동, 놓으면 원위치 복귀 |

### 3. 노드 인터랙션

#### 활성 노드 클릭
- 우측 사이드 패널 슬라이드 인
- 해당 노드와 연결된 다른 노드들 강조 (시각적 하이라이트)

#### 비활성 노드 클릭
- 클릭 가능
- 사이드 패널에 노드 정의만 표시 (자료 목록 비어 있음)

#### 빈 공간 클릭
- 사이드 패널 슬라이드 아웃
- 강조 해제

### 4. 우측 사이드 패널

#### 활성 노드 선택 시
```
┌────────────────────────────┐
│ [X]                        │
│                            │
│  노드명                     │
│  ──────────────────        │
│                            │
│  노드 정의/설명              │
│  (운영진 사전 정의 텍스트)    │
│                            │
│  ──────────────────        │
│                            │
│  관련 자료 (3개)            │
│  ┌──────────────────────┐  │
│  │ 자료 1 미리보기 ...   │  │
│  │ 자료 2 미리보기 ...   │  │
│  │ 자료 3 미리보기 ...   │  │
│  └──────────────────────┘  │
│  ↑ 섹션 클릭 시 팝업 호출    │
│                            │
│  ──────────────────        │
│                            │
│  연결 관계                   │
│  [기반 선택][맥락][확장][대조] │
│  기반 개념                   │
│  A, B, C, ...               │
│  [더보기]                   │
│                            │
└────────────────────────────┘
```

#### 비활성 노드 선택 시
```
┌────────────────────────────┐
│ [X]                        │
│                            │
│  노드명                     │
│  ──────────────────        │
│                            │
│  노드 정의/설명              │
│                            │
│  ──────────────────        │
│                            │
│  관련 자료 (0개)            │
│  (아직 활성화되지 않음)      │
│                            │
│  ──────────────────        │
│                            │
│  연결 관계                   │
│  [기반 선택][맥락][확장][대조] │
│  기반 개념                   │
│  A, B, C, ...               │
│  [더보기]                   │
│                            │
└────────────────────────────┘
```

#### 사이드 패널 표시 정보

| 항목 | 활성 노드 | 비활성 노드 |
|---|---|---|
| 노드명 | ⭕ | ⭕ |
| 노드 정의/설명 | ⭕ | ⭕ |
| 활성화시킨 자료 목록 (최신순) | ⭕ | 비어 있음 |
| 관계 토글 및 선택 관계 연결 노드 | ⭕ | ⭕ |

#### 자료 미리보기 섹션
- 최신순 정렬
- 자료 카드에는 자료 제목, 업로더, 업로드일만 표시한다.
- Evidence, AI 매칭 이유, 자료 본문 미리보기는 지식 구조 패널에서는 표시하지 않는다.
- **섹션 클릭 시 → 원본 자료 조회 팝업** (파생 화면 참조)

#### 관계 토글 및 선택 관계 연결 노드
- 연결 관계 영역은 우측 패널 카드 안에서 4개 관계 유형의 토글형 세그먼트 컨트롤로 표시한다: `기반 개념`, `맥락 개념`, `확장 개념`, `대조 개념`
- 관계 토글은 단일 선택만 허용하며, 선택된 토글은 채움/강조 상태로 표시하고 나머지는 기본 상태로 표시한다.
- 기본 선택값은 연결 노드가 있는 첫 번째 관계 유형이며, 연결 노드가 모두 없으면 `기반 개념`을 기본 선택한다.
- 사용자가 관계 토글을 선택하면 하단 목록은 해당 관계 유형의 연결 노드만 표시한다.
- 노드명만 쉼표 구분으로 나열하고 화면 너비 기준 줄바꿈한다.
- 연결 노드명은 클릭 가능하게 표시한다.
- 연결 노드명을 클릭하면 해당 노드로 그래프를 이동/강조하고 우측 사이드 패널도 해당 노드 상세로 전환한다.
- 선택된 관계에 연결 노드가 없으면 "연결된 노드가 없습니다" 빈 상태 문구를 표시한다.
- "더보기" 버튼은 선택된 관계 유형의 노드가 패널 내 기본 노출 수를 초과할 때만 표시하며, 클릭 시 해당 관계 유형의 전체 노드명을 노출한다.

## 기능 요구사항

### 그래프 초기 상태
- 스터디 생성 직후: 선택한 온톨로지 템플릿의 전체 노드가 **회색 반투명**으로 표시
- 노드 위치는 자동 레이아웃 (force-directed 등)

### 활성화 상태 갱신
- 자료 업로드 → AI 추출 → 검토 확정 시 자동 반영
- 활성화 강도는 검토 확정된 자료 수에 비례

### 주차별 상태 구분
- 주차 필터 선택 시 해당 주차에 처음 활성화된 노드와 기존 활성 노드에 자료가 추가된 보강 노드를 모두 강조한다.
- 신규 활성 노드와 보강 노드는 범례와 시각 스타일로 구분한다.
- 선택 주차에 신규/보강 노드가 없으면 그래프는 유지하되 해당 주차 데이터 없음 안내를 표시한다.

### 노드 검색
- 노드명 검색은 주차 필터와 독립적으로 동작한다.
- 검색 대상은 선택한 온톨로지 템플릿의 전체 노드다.
- 검색 결과 선택 시 주차 필터는 유지하되 검색 선택 노드를 우선 포커스하고 우측 패널을 연다.

### 종료된 스터디 처리
- 그래프 인터랙션 가능 (조회만)
- 사이드 패널의 자료 클릭 가능 (다운로드 가능)
- 새 자료 추가 불가 (다른 탭에서 차단)

## 디자인 가이드

### 시각 우선순위
1. 3D 그래프 (메인 시각 요소)
2. 사이드 패널 (선택 시)
3. 상단 필터 (보조)

### 색상 정책
- **활성 노드**: 단일 색상 (서비스 메인 컬러 또는 강조 색)
- **비활성 노드**: 회색 반투명 (낮은 채도)
- **강조 노드**: 테두리 강조 또는 글로우
- **관계선**: 흐린 회색 (노드보다 약하게)

### 사이드 패널 디자인
- 우측에서 슬라이드 인/아웃 애니메이션
- 그래프와 명확히 분리되는 배경

### 종료된 스터디
- 헤더에 "종료됨" 배지 추가
- 그래프 색상은 유지 (조회 가능 의미)

### MVP 제외
- 노드/관계 추가, 삭제, 수정 등 그래프 편집 기능은 제외한다.
- 검색 외 경로 기록, 즐겨찾기, 추천 다음 노드, 미니맵 등 고급 탐색 기능은 제외한다.

| 항목 | 내용 |
|---|---|
| 파일명 | 자료 제목 |
