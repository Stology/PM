## 화면 흐름 (전체 진입 경로)

```
[로그인] (01)
   ↓
   ├─ 초대 링크 로그인 성공 → [초대 수락 화면] (10)
   │                       └─ 참여하기 → [스터디 페이지 / 지식 구조 탭] (03)
   │
[홈] (02)
   ↓
   ├─ 스터디 카드 클릭 → [스터디 페이지] (07)
   │                       ├─ [지식 구조 탭] (03)
   │                       │    └ 노드 검색/노드 클릭 → 사이드 패널 → 원본 자료 조회 팝업
   │                       ├─ [자료 업로드 탭] (04)
   │                       │    └ 검토 필요 자료 클릭 → [AI 추출 결과 검토] (09)
   │                       ├─ [주차별 기록 탭] (05)
   │                       │    └ 노드 클릭 → 인라인 펼침 (자료 리스트)
   │                       ├─ [주차별 리포트 탭] (06)
   │                       └─ [질문함 탭] (08)
   │                            ├ 게시글 클릭 → 인라인 펼침 (질문 본문 + 답글)
   │                            ├ 질문 작성 버튼 → [질문 작성 모달]
   │                            └ 본인건 수정/삭제
   │
   ├─ + 카드 클릭 → [스터디 생성 모달] (02 내부)
   │                       └ 생성하기 → [초대 링크 생성 모달] (02 내부)
   │
   ├─ 내 할 일 자료 클릭 → [자료 상세 모달] → 검토 `REV001` 또는 빈 업로드 `UPL001`
   ├─ 내 할 일 질문함 클릭 → [질문함 상세 모달] → 해당 질문 펼침 `QNA001`
   ├─ 내 할 일 리포트 클릭 → [리포트 상세 모달] → 선택 스터디 최신 `RPT001`
   ├─ 팀 활동 행 클릭 → 지식 구조 반영은 `STD001`, 다른 팀원의 질문에 등록된 답글은 `QNA001`
   │
   └─ 사이드바 종료 스터디 → [스터디 페이지 조회 모드]

스터디장의 [스터디 페이지] → ⚙ 클릭 (07 내부)
   ├─ 초대 링크 → [초대 링크 생성 모달]
   ├─ 검토 인원 수 조정 → [검토 인원 수 조정 모달]
   ├─ 리포트 생성 시간 → [리포트 생성 시간 수정 모달]
   └─ 스터디 종료 → [스터디 종료 확인 모달] → [스터디 종료 안내 화면] → [홈]
```

## 파일 사용 가이드

### 와이어프레임 작성자

1. 화면 단위 작업 → 해당 번호 파일 1개만 보면 됨
2. 각 파일에 메인 화면 + 파생 화면이 모두 포함되어 자기 완결적
3. 헤더 명세는 `07-study-container.md`에 통합되어 있으므로 스터디 탭 작업 시 두 파일을 같이 참조

### 개발자

1. 화면별 진입점, 라우팅, 액션 명세 확인
2. 권한 정책은 각 파일 상단 "화면 개요"에 명시
3. 데이터 모델 / API 설계는 별도 문서(`stology-full-spec-v2.md`) 참조
