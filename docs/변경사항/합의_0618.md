## 합의 내용

1. `STD001`의 주목적은 스터디원의 학습 탐색이다.
2. 기존 PM 와이어프레임 형식은 유지한다.
3. 주차 필터는 선택 주차의 신규 활성 노드와 보강 노드를 모두 강조하고, 두 상태를 색상/배지/선 스타일 등으로 구분한다.
4. 우측 패널 카드의 연결 관계 영역은 `기반/맥락/확장/대조` 4개 토글형 세그먼트 컨트롤로 표시하고 단일 선택만 허용한다.
5. 관계 토글 하단의 연결 노드명은 클릭 가능하게 표시한다.
6. 연결 노드명을 클릭하면 해당 노드로 그래프를 이동/강조하고 우측 사이드 패널도 해당 노드 상세로 전환한다.
7. 관련 자료 미리보기 카드는 자료 제목, 업로더, 업로드일만 표시한다.
8. 관련 자료는 우측 패널과 원본 자료 조회 팝업 모두 최신순으로 정렬한다.
9. 상단 필터 영역에 노드명 검색을 추가한다.
10. 노드명 검색은 주차 필터와 무관하게 온톨로지 전체 노드를 대상으로 한다.
11. 검색 결과 노드를 선택하면 해당 노드로 그래프를 이동/강조하고 우측 사이드 패널을 해당 노드 상세로 전환한다.

## MVP 제외

- 노드/관계 추가, 삭제, 수정 등 그래프 편집 기능은 제외한다.
- 검색 외 경로 기록, 즐겨찾기, 추천 다음 노드, 미니맵 등 고급 탐색 기능은 제외한다.
