# 합의_0716_업로드주차 — 자료 업로드 주차 선택 제거

## 결정

- `UPL001 자료 업로드` 화면에서 주차 선택 입력과 인터랙션을 제거한다.
- 파일 업로드와 텍스트 직접 입력 모두 자료 제목만 필수로 유지하고, 자료 설명은 선택으로 유지한다.
- 등록한 자료는 사용자가 주차를 지정하지 않으며, 등록 시점의 스터디 **현재 진행 주차**에 자동 귀속한다.
- 주차를 탐색하는 기능은 지식 구조, 주차별 기록, 주차별 리포트 화면에만 유지한다.

## 검증 기준

- 자료 업로드 화면에 주차 드롭다운이나 주차 선택 인터랙션이 없어야 한다.
- 주차 미선택 검증은 없어야 하며, 입력 방식별 파일/본문과 자료 제목의 필수 검증은 유지되어야 한다.
- Notion, Figma, 로컬 문서가 현재 진행 주차 자동 귀속 정책을 동일하게 설명해야 한다.
