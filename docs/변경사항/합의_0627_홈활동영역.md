# 합의_0627_홈활동영역 — 내 할 일/팀 활동 구조 정리

> **후속 변경(2026-07-17)**: 팀 활동 표시 행과 표시 구조는 `합의_0717_팀활동행종류.md`로 대체한다. 기존 HOM001 우측 패널과 `전체 스터디` 선택 UI를 유지하고 별도 상세 모달 없이, `지식 구조 반영`, `다른 팀원의 질문에 등록된 답글` 두 종류만 행으로 표시한다.

## 변경 배경

- 기존 홈 하단은 `개인 활동 로그`와 `팀원 활동 로그`로 구성되어 있었으나, 개인 활동 이력은 사용자의 다음 행동으로 이어지는 가치가 낮다.
- 홈은 과거 기록보다 사용자가 바로 처리하거나 확인해야 할 항목과 팀의 최근 학습 흐름을 보여주는 진입 허브 역할을 우선한다.

## 합의 내용

1. 홈 하단 좌측 영역은 `개인 활동 로그`가 아니라 `내 할 일`로 변경한다.
2. 홈 하단 우측 영역은 `팀원 활동 로그`가 아니라 `팀 활동`으로 변경한다.
3. `내 할 일`은 로그인 사용자가 직접 처리하거나 확인해야 하는 항목만 표시한다.
4. `팀 활동`은 사용자가 참여 중인 진행 스터디에서 본인을 제외한 팀원 활동을 표시한다.
5. 참여 스터디가 여러 개일 수 있으므로 `팀 활동` 우측 상단에 스터디 선택 드롭다운을 제공한다.
6. 스터디 선택 드롭다운의 기본값은 `전체 스터디`이며, 옵션은 `전체 스터디`와 참여 중인 진행 스터디 목록이다.
7. 스터디를 1개만 참여 중이면 드롭다운은 숨김 처리한다.
