# Stology 화면설계서

> 산출물 목적: Figma `stology_wf`의 Mid-Fi 와이어프레임을 기준으로, 사용자가 서비스를 이용하는 흐름과 화면별 인터랙션을 PM/디자이너/개발자가 같은 기준으로 이해할 수 있게 정리한다.

---

## 0. 작성 기준

### 0-1. 화면설계서 구성

각 화면은 다음 3요소를 함께 관리한다.

| 구성 | 설명 |
|---|---|
| 화면 정보 | 화면 이름, 화면 ID, 메뉴 위치/뎁스, 접근 권한 |
| 와이어프레임 | Figma 프레임 내 번호 콜아웃으로 UI 요소 표시 |
| 상세 설명 | 번호별 액션, 이동 화면, 팝업 조건, 엠티/에러/상태 케이스 |

### 0-2. 브랜드 에셋 반영 기준

`Brand_Assets/` 분석 결과, Stology의 시각 언어는 아래 기준으로 정리한다.

| 항목 | 반영 내용 |
|---|---|
| Primary | `#5B5CF6` |
| Sub | `#C7C9FF` |
| Text | `#1F2430` |
| Background | `#F5F6FF` |
| 캐릭터 | `Stolo`: 지식 네비게이터, 학습 도우미 |
| 그래픽 모티프 | 노드, 연결선, 지식 그래프, 성장/검증 |
| 키워드 | 연결, 성장, 신뢰, 협업, 기록 |

> 와이어프레임은 Mid-Fi 단계이므로 최종 Hi-Fi 이미지 중심 디자인이 아니라, 브랜드 색상/모티프/캐릭터 사용 위치를 화면 설계 수준에서 정의한다.

### 0-3. Notion/Figma/docs 통합 기준

- 기준 원본: Notion `기능 명세서`(`3651d8db-b5f5-80b2-b15a-f1d62ce1acfe`).

## 2. 홈 및 스터디 생성

### 2-1. 홈 화면 조회

| 항목 | 내용 |
|---|---|
| 화면 이름 | 홈 |
| 화면 ID | `HOM001` |
| 뎁스 | 로그인 → 홈 |
| 권한 | 로그인 사용자 |
| Figma 프레임 | `02-01. 홈 HOM001`, `02-03. 홈 빈 상태 HOM001`, `02-08. 활동 영역 예외 HOM001-0300` |

#### 와이어프레임 번호

| 번호 | UI 요소 | 상세 설명 |
|---|---|---|
| 1 | 새 스터디 생성 카드 | 클릭 시 스터디 생성 모달을 호출한다. 진행 중 스터디가 없어도 항상 노출한다. |
| 2 | 내 할 일 | 현재 3개 섹션을 유지한다. 자료는 `검토 N · 재업로드 N`, 질문함은 `질문 N · 답글 N`, 리포트는 `N개 스터디의 최신 리포트`로 요약하며 섹션 클릭 시 각각 `HOM001-0400/0500/0600`을 연다. |
| 3 | 스터디 생성 모달 | 필수값 입력 후 `생성하기` 클릭 시 초대 링크 모달로 자동 전환한다. |
| 4 | 팀 활동 | HOM001 우측의 기존 패널과 `전체 스터디` 선택 UI를 유지한다. 선택한 스터디 단위의 `지식 구조 반영`, `다른 팀원의 질문에 등록된 답글` 두 종류만 같은 패널의 최신순 행 목록으로 표시한다. 별도 상세 모달은 제공하지 않으며, 행 전체 클릭 시 각각 해당 `STD001`, `QNA001`로 바로 이동한다. 자료 업로드와 AI 후보 추출 완료는 제외한다. |

#### 케이스별 화면

| 케이스 | 조건 | 결과 화면/상태 |
|---|---|---|
| 참여 스터디 있음 | 진행 중 스터디 1개 이상 | 스터디 카드 가로 목록 + 새 스터디 생성 카드 |
| 참여 스터디 없음 | 진행 중 스터디 0개 | 빈 안내 문구 + 새 스터디 생성 카드 |
| 종료 스터디 있음 | 종료된 스터디 존재 | 사이드바 `지난 활동` 아래에 표시 |
| 내 할 일 합계 0개 | 자료/질문함/리포트 모두 0개 | 3개 섹션과 0 카운트를 유지하고 상세 모달에서 `내용 없음` 표시 |
| 팀 활동 없음 | 필터에 해당하는 활동 0개 | `아직 팀 활동이 없습니다.` 표시 |

#### 활동 상세 공통 예외

- 대상 삭제·상태 변경·권한 상실 시 이동을 막고 토스트를 표시한다.
- 현재 홈/모달에 머문 채 목록을 새로고침해 해당 행을 제거하고 홈·필터 숫자를 다시 계산한다.
- 선택 필터에 항목이 없으면 `내용 없음`을 표시한다.
